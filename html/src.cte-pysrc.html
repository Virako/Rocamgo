41  """ 
42  FEATURES OF FILES .SGF 
43  ==================================== 
44   
45  * AB: Add Black: locations of Black stones to be placed on the board prior to the first move. 
46  * AW: Add White: locations of White stones to be placed on the board prior to the first move. 
47  * AN: Annotations: name of the person commenting the game. 
48  * AP: Application: application that was used to create the SGF file (e.g. CGOban2,...). 
49  * B: a move by Black at the location specified by the property value. 
50  * BR: Black Rank: rank of the Black player. 
51  * BT: Black Team: name of the Black team. 
52  * C: Comment: a comment. 
53  * CP: Copyright: copyright information. See Kifu Copyright Discussion. 
54  * DT: Date: date of the game. 
55  * EV: Event: name of the event (e.g. 58th Honinbo Title Match). 
56  * FF: File format: version of SGF specification governing this SGF file. 
57  * GM: Game: type of game represented by this SGF file. A property value of 1 refers to Go. 
58  * GN: Game Name: name of the game record. 
59  * HA: Handicap: the number of handicap stones given to Black. Placment of the handicap stones are set using the AB property. 
60  * KM: Komi: komi. 
61  * ON: Opening: information about the opening (fuseki), rarely used in any file. 
62  * OT: Overtime: overtime system. 
63  * PB: Black Name: name of the black player. 
64  * PC: Place: place where the game was played (e.g.: Tokyo). 
65  * PL: Player: color of player to start. 
66  * PW: White Name: name of the white player. 
67  * RE: Result: result, usually in the format "B+R" (Black wins by resign) or "B+3.5" (black wins by 3.5 moku). 
68  * RO: Round: round (e.g.: 5th game). 
69  * RU: Rules: ruleset (e.g.: Japanese). 
70  * SO: Source: source of the SGF file. 
71  * SZ: Size: size of the board, non square boards are supported. 
72  * TM: Time limit: time limit in seconds. 
73  * US: User: name of the person who created the SGF file. 
74  * W: a move by White at the location specified by the property value. 
75  * WR: White Rank: rank of the White player. 
76  * WT: White Team: name of the White team.  
77   
78  """
